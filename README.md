# Protocol Zero

Open source builder focused on **OpenClaw**, agentic infrastructure, and security.

## Current Focus
- Contributing to the OpenClaw ecosystem
- Building China-side ecosystem and builder resources
- Exploring hardware and wearable workflows
- Improving security, signal quality, and reusable agent workflows

## Featured Work
- `openclaw`: contribution workspace aligned with the main project
- `awesome-openclaw-china-ecosystem`: China-side ecosystem map and resources
- `awesome-openclaw-hardware`: hardware, glasses, audio, and edge directions
- `OpenClaw-Cortex`: context quality and denoising layer for assistants
- `openclaw-security-radar`: security guidance for builders
- `openclaw-agent-packs`: reusable workflow packs for common use cases
